Rectangle MuPDFCore.MuPDFPage.GetBoundingBox ( BoxType  boxType,

)

Gets the specified bounding box from the page.

Parameters
boxTypeThe type of bounding box to get.
rescaleIf this is true, the bounding box is rescaled so that it is expressed in the same resolution units as the underlying document. If this is false, the raw value is returned (at 72 dpi).
Returns
A Rectangle corresponding to the specified bounding box.
Exceptions
MuPDFExceptionThrown if the bounding box cannot be computed or if another error occurs.
